## Tujuan Yang Harus Dicapai

Para users harus bisa:

- Melihat layout yang baik dan sesuai dengan ukuran masing-masing device mereka
- Melihat efek visual "hover" untuk semua interactive elements di halaman web
- Membuka galeri yang berisi foto lain dari produk tersebut by clicking foto terbesar produk
- Merubah foto terbesar produk dengan mengklik thumnail kecil foto produk
- Menambah item ke keranjang
- Melihat isi keranjang belanjaan dan menghapus item di dalamnya
